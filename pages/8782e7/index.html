  2. 在py文件中，使用环境变量

    1. 步骤

      1. 引入os 和 dotenv 模块
      2. 用load_dotenv方法，加载配置文件
      3. 通过key 方式，调用环境变量
    2. #1. 引入os 和 dotenv 模块
import os
import dotenv
# 2. 调用load_dotenv方法，加载配置文件

print(os.getenv('OPEN_GPT_URL'))
