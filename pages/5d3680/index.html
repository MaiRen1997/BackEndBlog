# 高阶函数

# 定义

当一个函数的参数是函数或者返回值是函数，那该函数就是高阶函数

# 作用

  1. 代码复用性高：可以把行为”独立出去“，传入不同函数，实现不同逻辑
  2. 能让函数更灵活，更通用
  3. 高阶函数是：装饰器、闭包的基础

# 说明

# 入参是函数

  1. 如caller函数，因为入参是函数，所以caller函数是高阶函数

def printInfo(msg):
    print(msg)
    
def caller(f):
    print('函数调用了')
    f()
