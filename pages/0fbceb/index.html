匿名函数、闭包

# 匿名函数

  1. 使用场景： 当一个函数只用一次，只做一点点小事，使用匿名函数会更简洁

  2. 匿名函数，就是没有名字的函数，那么什么函数没有名字呢？这个就是我们以工作中经常用匿名函数 lambda，也叫一句话函数。

    def func(a,b):
    return a+b
print(func(3,4))
    1
    2
    3

  3. 那么接下来我们用匿名函数完成上面的需求：

    func = lambda a,b: a+b
print(func(3, 4))  # 7
    1
    2

  4. 语法:

    • 函数名 = lambda 参数:返回值

      • 如果有多个参数，使用逗号分隔

      • 如果没有参数，就不写

      • func = lambda x, y: x + y

func = lambda : 10 + 20
        1
        2
        3

  5. 说明

    1. 匿名函数只能写一行.且逻辑结束后直接返回数据
    2. 不能写代码块(if、for、while)，但是可以写条件表达式 ：'成年' if age > 18 else '未成年'
    3. 冒号右边必须是表达式，且只能写一个表达式
    4. 表达式结果自动作为返回值，不需要写return
    5. 此函数不是没有名字，他是有名字的，他的名字就是你给其设置的变量，比如func.
    6. lambda 是定义匿名函数的关键字，相当于函数的def.
    7. lambda 后面直接加形参，形参加多少都可以，只要用逗号隔开就行。
    8. 返回值在冒号之后设置，返回值和正常的函数一样,可以是任意数据类型。

  6. 现在有一个需求：你们写一个函数，此函数接收两个int参数，返回和值。

    func = lambda a,b,*args,sex= 'alex',c,**kwargs: kwargs
print(func(3, 4,c=666,name='alex'))  # {'name': 'alex'}
# 所有类型的形参都可以加，但是一般使用匿名函数只是加位置参数，其他的用不到。

    2
    3

  7. 几个小题目:

    1. 写匿名函数：接收一个可切片的数据，返回索引为0与2的对应的元素（元组形式）。

      func = lambda x:(x[0],x[2])
print(func('afafasd'))

    2. 写匿名函数：接收两个int参数，将较大的数据返回

      func = lambda x,y: x if x > y else y
print(func(3,100))

# 闭包

# 定义

  1. 闭包是嵌套在函数中的函数。
  2. 闭包必须是内层函数对外层函数的变量（非全局变量）的引用。

# 作用

  1. 保存局部信息不被销毁，保证数据的安全性。

# 应用

  1. 可以保存一些非全局变量但是不易被销毁、改变的数据。

  2. 装饰器。

    def make_averager():
    
    series = []
    def averager(new_value):
        series.append(new_value)
        total = sum(series)
        return total/len(series)
    return averager
avg = make_averager()
print(avg(100000))
print(avg(110000))
print(avg(120000))

  3. 上面代码就是闭包，series这个变量应该是make_averager()函数的局部变量， 它应该是随着make_averager()函数的执行结束之后而消失。 但是他没有，是因为此区域形成了闭包，series变量就变成了一个叫自由变量的东西， averager函数的作用域会延伸到包含自由变量series的绑定。 也就是说，每次我调用avg对应的averager函数 时，都可以引用到这个自用变量series，这个就是闭包
