# 总结

  • 可迭代对象：一般为数据容器，且保留 迭代接口__iter__。
  • 迭代器：类，实现了__iter__和 __next__接口的类。
  • 生成器：函数，通过yield遍历元素。
  • 1、__iter__方法是留给程序获得 迭代对象 iter()
  • 2、获得迭代对象后，通过__next__ 或next()遍历元素。

# 定义
