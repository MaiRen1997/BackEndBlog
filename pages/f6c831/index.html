从零创建项目

# 第一步：安装 uv

# macOS / Linux

curl -LsSf https://astral.sh/uv/install.sh | sh
1

# 验证安装

uv --version
1

# 第二步：创建项目目录并初始化

# 1. 创建项目目录
mkdir my-awesome-project
cd my-awesome-project

# 2. 初始化项目
uv init

# 第三步：创建虚拟环境

uv init 只是生成了配置文件，还没有创建虚拟环境。创建虚拟环境：

# 创建虚拟环境（默认在 .venv 目录）
uv venv

# 或者指定 Python 版本
uv venv --python 3.12

# 第四步：同步依赖

虽然还没有添加依赖，但执行 uv sync 可以确保环境同步：

uv sync
1

这会：

  1. 检查虚拟环境是否存在
  2. 根据 pyproject.toml 安装依赖（目前为空）
  3. 更新 uv.lock 文件

# 第五步：添加项目依赖

假设我们要创建一个 Web API 项目，添加 FastAPI 和 Uvicorn：

# 添加运行时依赖
uv add fastapi uvicorn

# 添加开发依赖（用于测试和代码格式化）
uv add pytest black --dev

# 第六步：编写代码

修改 main.py，添加一个简单的 FastAPI 应用：

python

# main.py
from fastapi import FastAPI

app = FastAPI(title="My Awesome API", version="0.1.0")

@app.get("/")
def read_root():
    return {"message": "Hello, World!"}

@app.get("/health")
def health_check():
    return {"status": "healthy"}

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)

# 第八步：添加测试

创建测试文件 test_main.py：

python

# test_main.py
from fastapi.testclient import TestClient
from main import app

client = TestClient(app)

def test_read_root():
    response = client.get("/")
    assert response.status_code == 200
    assert response.json() == {"message": "Hello, World!"}

def test_health_check():
    response = client.get("/health")
    assert response.status_code == 200
    assert response.json() == {"status": "healthy"}
