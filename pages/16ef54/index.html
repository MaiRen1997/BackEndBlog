概述-(基本)数据类型

# 概述

  1. Go 语言中数据类型分为：基本数据类型和复合数据类型
    1. 基本数据类型有：
      1. 整型
      2. 浮点型
      3. 布尔型
      4. 字符串
    2. 复合数据类型有：
      1. 数组
      2. 切片
      3. 结构体
      4. 函数
      5. map
      6. 通道（channel）
      7. 接口
