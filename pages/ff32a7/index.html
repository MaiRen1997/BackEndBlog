# 表达式

执行后能得到值的代码，就是表达式（表达式最终会形成一个值，可以写在任何需要值的地方）

a = 3 + 5
a2 = 5 > 3
a3 = 'y' in 'python'
a4 = len('hello')

# 条件表达式

根据条件的真假，在两个结果中二选一的表达式（又称：三元表达式、三目运算符）

值1 if 条件 else 值2
1
text = '成年' if age > 18 else "未成年"
