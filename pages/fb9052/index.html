# 注意

  1. 在golang中， ++ 和 -- 只能单独使用，错误的写法如下

    var i int = 8
var a int
a = i++  // 错误，i++只能单独使用
a = i--  // 错误，i--只能单独使用

    4

  2. 同时在golang中，没有 ++i这样的操作

    var i int = 1
++i  // 错误
    1
    2

  3. 正确的写法

    var i int = 1
i++ //正确
