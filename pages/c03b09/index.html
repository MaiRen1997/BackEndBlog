# 仅限关键字参数

  1. 仅限关键字参数是python3x更新的新特性，
  2. 他的位置要放在*args后面，kwargs前面（如果有kwargs），也就是默认参数的位置，
  3. 它与默认参数的前后顺序无所谓，它只接受关键字传的参数

# 形参位置总结

位置参数，*args，默认参数，仅限关键字参数，**kwargs

# 高阶函数

# 匿名函数、闭包

# 匿名函数

  • 函数名 = lambda 参数:返回值
  • 参数多个就逗号隔开，没有，就一个都不屑，保留冒号

# 闭包

  1. 闭包构成条件三要素
    1. 函数嵌套
    2. 内部函数作为外部函数返回值

# 装饰器

# 基础装饰器

  1. 保证闭包三要素
  2. 外层函数的形参是func
  3. 内部函数调用func，内部函数的形参式 func的形参

# 带返回值装饰器

  1. 内部函数只需要将结果return即可

# 被装饰函数带参数

  1. 外部函数，形参接收被装饰方法
  2. 内部函数，接收被装饰方法的变量

# 标准装饰器

  1. 使用@+函数名语法糖， @timer 等价于 # home = timer(home)

# 带参数装饰器

  1. 相对于不带参数装饰器，外面多一层级函数，用于接收装饰器参数

# 取消装饰器

  1. 需要额外在内部方法上，使用@wraps(func)
  2. 调用时，使用原方法.__wrapped__(参数)即可
