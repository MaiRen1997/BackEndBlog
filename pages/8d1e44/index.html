# 创建表

  1. 使用__tablename__定义表名

  2. 使用primary_key=True定义主键

  3. Field(default_factory=lambda: uuid4().hex)

    1. default：提供一个固定的默认值

    2. default_factory：提供一个生成默认值的函数，每次需要时动态计算

    3. sa_column_kwargs={"server_default": text("CURRENT_TIMESTAMP")}

      1. 这是数据库级别的默认值，不同于 default 或 default_factory

    4. 定义成自增id

      1. 方式一：类型为int，并且primary_key =true

        # SQLModel 会自动识别 int 类型的主键为自增
id: int | None = Field(default=None, primary_key=True)
        1
        2

      2. 方式二：显式指定 autoincrement

        id: int | None = Field(
    default=None, 
    primary_key=True,
    sa_column_kwargs={"autoincrement": True}  # 显式指定自增
)
        1
        2
        3
        4
        5

    5. id的其余常用参数

      1. default="hello"
      2. "unique": True, # 唯一约束
      3. "index": True, # 索引
      4. "nullable": False # 不能为空

    6. 使用时间戳

      1. text('CURRENT_TIMESTAMP')
      2. server_default表示：数据库默认值
      3. onupdate表示： 更新时的值，如果设置为时间戳，表示更新时，默认设置为当前时间戳
class DialogTable(SQLModelSerializable, table=True):
    __tablename__ = "dialog"

    dialog_id: str = Field(default_factory=lambda: uuid4().hex, primary_key=True)
    name: str = Field(description='对话绑定的Agent的名称')
    agent_id: str = Field(description='对话Dialog绑定Agent的ID')
    agent_type: str = Field(default="Agent", description="对话Dialog绑定Agent, MCPAgent or Agent")
    user_id: str = Field(description='对话Dialog的用户ID')
    update_time: Optional[datetime] = Field(sa_column=Column(
            DateTime,
            nullable=False,
            server_default=text('CURRENT_TIMESTAMP'),
            onupdate=text('CURRENT_TIMESTAMP')
        ),
        description="修改时间"
    )
    create_time: Optional[datetime] = Field(
        sa_column=Column(
            DateTime,
            nullable=False,
            server_default=text('CURRENT_TIMESTAMP')
        ),
        description="创建时间"
    )

# sa_column与sa_column_kwargs

# 区别

  1. sa_column后面跟Column对象
  2. sa_column_kwargs后面跟键值对，dict类型
updated_at: datetime = Field(
    sa_column=Column(
        DateTime,
        nullable=False,
        server_default=text('CURRENT_TIMESTAMP'),
        onupdate=text('CURRENT_TIMESTAMP')
    )
)

updated_at: datetime = Field(
    sa_column_kwargs={
        "nullable": False,
        "server_default": text('CURRENT_TIMESTAMP'),
        "onupdate": text('CURRENT_TIMESTAMP')
    }
)

# server_default与default与default_factory

# 区分

特性 default default_factory server_default
执行位置 Python 端 Python 端 数据库端
值类型 固定值 可调用对象（函数/类） SQL 表达式/文本
执行时机 创建实例时 创建实例时（调用函数） INSERT 语句执行时
可变性 不可变（固定值） 可变（每次重新计算） 数据库决定
commit 前可见 ✅ 是 ✅ 是 ❌ 否（需 refresh）
可移植性 高 高 低（依赖数据库）
典型场景 静态默认值 动态默认值 数据库函数/约束

# 设置当前时间默认值

  1. default和default_factory，一般使用python代码，生成当前时间，作为值
  2. server_default、onupdate这类的数据库端属性，一般使用text('CURRENT_TIMESTAMP')
