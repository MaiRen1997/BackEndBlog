## 连接数据库

使用`sqlmodel`的`create_engine`
或者`sqlalchemy.ext.asyncio`的`create_async_engine`创建连接

```python
from sqlmodel import SQLModel, create_engine, text
from sqlalchemy.ext.asyncio import create_async_engine


engine = create_engine(
    url=app_settings.mysql.get('endpoint'),
    pool_pre_ping=True, # 连接前检查其有效性
    pool_recycle=3600, # 每隔1小时进行重连一次
    connect_args={
        "charset": "utf8mb4",
        "use_unicode": True,
        "init_command": "SET SESSION time_zone = '+08:00'"
    }
)


async_engine = create_async_engine(
    url=app_settings.mysql.get('async_endpoint'),
    pool_pre_ping=True,  # 连接前检查其有效性
    pool_recycle=3600,  # 每隔1小时进行重连一次
    connect_args={
        "charset": "utf8mb4",
        "use_unicode": True,
        "init_command": "SET SESSION time_zone = '+08:00'"
    }
)
```

## 不同的url值

| 数据库类型        | URL 格式示例                                  | 说明                                                     |
| :---------------- | :-------------------------------------------- | :------------------------------------------------------- |
| **SQLite (文件)** | `sqlite:///database.db`                       | `database.db` 是你的数据库文件名。                       |
| **SQLite (内存)** | `sqlite://`                                   | 数据库只在内存中运行，程序关闭后数据会丢失，常用于测试。 |
| **PostgreSQL**    | `postgresql://user:password@localhost/dbname` | 需要安装 `psycopg2` 或 `asyncpg` 驱动。                  |
| **MySQL**         | `mysql://user:password@localhost/dbname`      | 需要安装 `pymysql` 或 `mysqlclient` 驱动。               |
| **SQL Server**    | `mssql+pyodbc://user:password@host/dbname`    | 需要安装 `pyodbc` 驱动。                                 |
