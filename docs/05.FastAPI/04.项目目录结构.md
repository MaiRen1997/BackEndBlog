## 项目结构示例

```text
fastapi-project/
├── main.py              # 应用入口
├── core/
│   ├── config.py        # 配置管理
│   ├── security.py      # 安全认证
│   └── database.py      # 数据库连接
├── models/              # 数据库模型
├── schemas/             # Pydantic模型
├── routers/             # 路由模块
├── services/            # 业务逻辑
└── tests/               # 单元测试
```
