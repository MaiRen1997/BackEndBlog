## 引入文件路径

### 背景

在引入文件路径时，Python是以运行python命令时，所在的目录为base_url进行拼接的
所以在写文件路径时，如果用户操作的目录不对，就会造成文件引入不成功

### 解决方案

可以使用Path(`__file__`)来获取当前文件的绝对路径

然后通过`.parent`或者`[数字]`，来往上寻找父级目录

```python
from pathlib import Path
from omegaconf import OmegaConf
config_file = Path(__file__).parents[2]/'conf'/ 'app_config.yaml

conf = OmegaConf.load(config_file)
print(typq(conf))
```
