### from ...  import 导入

#### 导入方法

#### 一行导入多个

```python

```

#### 控制哪些方法可以导出

```python
__all__=['money','read1'] #这样在另外一个文件中用from spam import *就这能导入列表中规定的两个名字
```

### py文件的两种方式

#### 当作脚本

1. 一个文件就是整个程序，用来被执行

2. `__name__` 等于`__main__`

3. ```python
   if __name__ == '__main__':  
      # 在模块文件中测试read1()函数
      # 此模块被导入时 __name__ == tbjx 所以不执行
      read1()
   ```

#### 当作模块

1. `__name__`等于模块名

#### 模块的搜索路径

1.  **内存中已经加载的模块->内置模块->sys.path路径中包含的模块**
