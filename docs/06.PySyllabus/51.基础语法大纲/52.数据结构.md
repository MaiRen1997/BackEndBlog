3. 新增
   1. append 最后追加

   2. insert 任意位置追加

   3. extend  迭代着追加

   4. ```python
      # append 追加，给列表的最后面追加一个元素
      l = [1, 2, 'a']

      print(l) # [1, 2, 'a', 666]
      
      # insert  插入在列表的任意位置插入元素
      l = [1, 2, 'a']

      l = [1, 2, 'a']

      ```

6. 删
   1. pop

   2. remove

   3. clear

   4. del + 切片(步长)/下标 实现删除

   5. ```python
      # pop  通过索引删除列表中对应的元素，该方法有返回值，返回值为删除的元素
      l = ['张三', 'alex', 'WuSir', '女神']
      ret = l.pop(1)
      print(ret,l) # alex ['张三', 'WuSir', '女神']
      
      
      # remove  通过元素删除列表中该元素
      l = ['张三', 'alex', 'WuSir', '女神']
      l.remove('alex')
      print(l) # ['张三', 'WuSir', '女神']
      
      # clear 清空列表
      l = ['张三', 'alex', 'WuSir', '女神']
      l.clear()
      print(l) # []  
      
      
      # del
      #按照索引删除该元素
      l = ['张三', 'alex', 'WuSir', '女神']
      del l[2]
      print(l) # ['张三', 'alex', '女神']
      
      
      # 切片删除该元素
      l = ['张三', 'alex', 'WuSir', '女神']
      del l[1:]
      print(l) # ['张三']
      ```
