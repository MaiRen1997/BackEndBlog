## 自定义异常

1. 继承Exception

2. 定义init和str两个魔法方法

3. ```python
   class MyError(Exception):
       def __init__(self, value):
           self.value = value
   
       def __str__(self):
           return repr(self.value)
   
   try:
       raise MyError(1)
   except MyError as e:
       print("触发自定义异常:", e.value)
   ```

# 迭代器生成器

## 迭代器

### 如何判断是否可迭代

1. 有`__iter__`方法的对象是可迭代对象

2. 有`__iter__`和`__next__`两个方法的对象，是迭代器

3. 判断是否有iter方法

   1. ```python
      s1 = 'alex'
      i = 100
      print('__iter__' in dir(i))  # False
      print('__iter__' in dir(s1))  # True
      ```

### 什么是迭代器

`迭代器是一个可以记住遍历的位置的对象`

`list 、 tuple 、 dict 、 set 、 str`

### 可迭代对象与迭代器的对比

#### 可迭代对象

1. 有`__iter__`方法

#### 迭代器

1. 有`__iter__`、`__next__`方法
2. 使用next()或者`__next__`方法访问内部元素

### 自定义迭代器

```python
class Reverse:
    """对一个序列执行反向循环的迭代器。"""

    def __init__(self, data):
        self.data = data
        self.index = len(data)

    def __iter__(self):
        return self

    def __next__(self):
        if self.index == 0:
            raise StopIteration
        self.index = self.index - 1
        return self.data[self.index]

rev = Reverse([2, 3, 5, 7, 11, 13, 17, 19])
iter(rev)
for char in rev:
    print(char)
    
    
    
# 使用next
obj = iter(rev)
print(next(obj))  # 输出: 19
```

## 生成器

### 基础使用

#### 创建方式

1. 通过生成器函数(yield)

2. 一个yield对应一个next，next超过yield数量，就会报错

3. 可以使用`gener.__next__()`或者`next(gener)`来一步步执行

   ```python
   def func():
        print("111")
        yield 222
   gener = func() # 这个时候函数不会执⾏. ⽽是获取到⽣成器
   ret = gener.__next__() # 这个时候函数才会执⾏
   
   
   
   next(gener)
   ```

4. 通过生成器推导式

   1. 生成器表达式和列表推导式的语法上一模一样,只是把[]换成()就行了

   2. ```python
      # 获取1-100内能被3整除的数
      gen = (i for i in range(1,100) if i % 3 == 0)
      for num in gen:
          print(num)
      ```


#### List与生成器的区别

1. List是直接把元素全部生成，占用内存。
2. 生成器是生成一个，用一个，非常的节省内存，而且还可以保留上次的位置。

### Send方法

#### send和next()区别:

1. 相同点：

   send 和 next()都可以让生成器对应的yield向下执行一次。

   都可以获取到yield生成的值。

2. 不同点：

   第一次获取yield值只能用next不能用send（可以用send(None)）。

   send可以给上一个yield置传递值。

#### send的使用

```python
# 而使用send这个方法是可以的。
def gen(name):
    print(f'{name} ready to eat')
    while 1:
        food = yield 222
        print(f'{name} start to eat {food}')
dog = gen('alex')
next(dog)  # 第一次必须用next让指针停留在第一个yield后面
# 与next一样，可以获取到yield的值
ret = dog.send('骨头')
print(ret)
```

### yield from

1. 如果是yield写法

   ```python
   def func():
       lst = ['卫龙','老冰棍','北冰洋','牛羊配']
       yield lst
   g = func()
   print(g)
   print(next(g))  # 只是返回一个列表 ['卫龙', '老冰棍', '北冰洋', '牛羊配']
   ```

2. 如果是yield from 写法

   ```python
   def func():
       lst = ['卫龙','老冰棍','北冰洋','牛羊配']
       yield from lst
   g = func()
   print(g)
   # 他会将这个可迭代对象(列表)的每个元素当成迭代器的每个结果进行返回。
   print(next(g))
   print(next(g))
   print(next(g))
   print(next(g))
   
   卫龙
   老冰棍
   北冰洋
   牛羊配
   ```
