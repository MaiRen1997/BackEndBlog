## 参数的解包和打包

*`号用在形参上，表示打包，*号用在实参上，表示解包`

1. 形参
   1. *args： 打包所有的位置参数(会形成一个元组)
   2. **kwargs: 打包所有的关键字参数(会形成一个字典)
2. 实参
   1. *变量名： 将元组拆解成一个个独立的位置参数
   2. **变量名：将字典拆解成一个个key=value形式的关键字参数

```python
s1 = 'alex'
l1 = [1, 2, 3, 4]
tu1 = ('武sir', '太白', '女神',)
def func(*args):
    print(args) # ('a', 'l', 'e', 'x', 1, 2, 3, 4, '武sir', '太白', '女神')
func(*s1,*l1,*tu1)


s1 = 'alex'
l1 = [1, 2, 3, 4]
tu1 = ('武sir', '太白', '女神',)
def func(*args):
    print(args) # ('alex', [1, 2, 3, 4], ('武sir', '太白', '女神'))
func(s1,l1,tu1)
```

### 自定义迭代器

```python
class Reverse:
    """对一个序列执行反向循环的迭代器。"""

    def __init__(self, data):
        self.data = data
        self.index = len(data)

    def __iter__(self):
        return self

    def __next__(self):
        if self.index == 0:
            raise StopIteration
        self.index = self.index - 1
        return self.data[self.index]

rev = Reverse([2, 3, 5, 7, 11, 13, 17, 19])
iter(rev)
for char in rev:
    print(char)
    
    
    
# 使用next
obj = iter(rev)
print(next(obj))  # 输出: 19
```

## 装饰器

### 基础装饰器

1. 保证闭包三要素
2. 外层函数的形参是func
3. 内部函数调用func，内部函数的形参式 func的形参

### 带返回值装饰器

1. 内部函数只需要将结果return即可

### 被装饰函数带参数

1. 外部函数，形参接收被装饰方法
2. 内部函数，接收被装饰方法的变量

### 标准装饰器

1. 使用`@+函数名语法糖`， @timer 等价于  # home = timer(home)

### 带参数装饰器

1. 相对于不带参数装饰器，外面多一层级函数，用于接收装饰器参数

### 取消装饰器

1. 需要额外在内部方法上，使用@wraps(func)
2. 调用时，使用`原方法.__wrapped__(参数)`即可
